This project is generated using Yeoman (http://yeoman.io/)

### Requirements
- gem
- npm (version > 1.2.10)
- grunt (`npm install -g grunt-cli`)

### Install
- `npm install`
- `bower install`
- `grunt serve`
- look your browser

### Publish on gh-pages
``` sh
grunt build
git subtree push --prefix dist origin gh-pages
```
